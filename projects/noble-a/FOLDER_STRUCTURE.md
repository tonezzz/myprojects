# Folder Structure

This document outlines the structure of the project folder `projects/noble-a/`, including the purpose of each folder and file.

## 1. `projects/noble-a/`
This is the main project directory for the Noble-A project.

### 2. `accounting/`
This subdirectory contains all accounting related files and reports.

#### 3. `2025/`
This folder contains all the accounting records for the year 2025.

##### 4. `monthly_report/`
This folder is used to store monthly reports. Each month's reports are organized into their respective folders.

###### 5. `2025-07/`
- **README.md**: This file contains information about the content and context of the files in this month’s reports.
  - Purpose: To provide a summary and insights regarding the data contained in this folder.
- **EXTRACTED_DATA.md**: This file includes the relevant data extracted for the month.
  - Purpose: To present raw data in a structured manner for analysis and review.
- **[PDF files]**: Contains all PDF files generated for this month’s reporting.

###### 6. `2025-08/`
- **README.md**: Similar to July's README, this provides insights regarding the August content.
  - Purpose: To summarize and clarify the information present in the folder.
- **EXTRACTED_DATA.md**: This file contains the relevant extracted data for August's activities.
  - Purpose: To catalog all pertinent data in an organized fashion.
- **[PDF files]**: Contains all the necessary PDF documents for this month’s reporting.

## Summary
This folder structure helps in organizing and managing accounting materials efficiently, ensuring that both reports and extracted data are neatly categorized by month and accessible for auditing or review purposes.
